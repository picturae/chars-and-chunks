hotkey funtion

  • when the function is called the callback and the description are registered with the single charcter
  • when the single charcter is pressed the callback executes
  • when the context of the single charcter is removed the callback is not accessable forever

    barcode funtion

  • when a barcode is encountered the callback processes the barcode
  • when the context of the callback is removed the callback is not accessable forever
